    1. What is Semantic HTML?
	The use of tags with specific meaning.
    2. What is HTML used for? 
	Displaying information in a logical way on a webpage.
    3. What is an attribute and where do we put it? 
	attributes help identify and provide locations for certain tags.
    4. What is the h1 tag used for? How many times should I use it on a page?
	Main heading/Title.  Should only be used 1 time / page.
    5. Name two tags that have required attributes
	anchor tag (href), and img tag (src).
    6. What do we put in the head of our HTML document? 
	The website title and information to help search engines classify our site/page.  locations of external stylesheets, fonts, and such also go here.
    7. What is an id? 
	a way to identify 1 specific element per page.
    8. What elements can I add an id to? 
	any
    9. How many times can I use the same id on a page? 
	1 time
    10. What is a class? 
	a way to group of elements for styling purposes.

	any/all/none/some.
    12. How many times can I use the same class on a page? 
	As many as you need to.

	Absolute positioning allows you to place your element precisely where you want it. Absolute positioning is done relative to the
	first relatively (or absolutely) positioned parent element. In the case when there is no positioned parent element, the element
	 that has position set to absolute will be positioned related directly to the HTML element (the page itself).
